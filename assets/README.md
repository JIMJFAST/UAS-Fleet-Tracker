# Assets Folder

Place your app icon here:

- `icon.ico` - Windows icon (256x256 recommended)
- `icon.icns` - macOS icon
- `icon.png` - Linux icon (512x512 recommended)

## Creating Icons

1. Create a 512x512 PNG image
2. Use online converter: https://convertico.com/
3. Save the .ico file here as `icon.ico`

## Default Icon

If no icon is provided, Electron will use its default icon.
